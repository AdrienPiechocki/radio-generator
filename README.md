# Radio Genetrator

Generate radio audios (with subtitles) in French.

## Usage

```bash
./run.sh "news" "rss feed url"     # gen news
./run.sh "meteo" "city"		         # gen forecast for today
./run.sh "meteo_demain" "city"     # gen forecast for tomorrow
./run.sh "podcast" "theme"         # gen announce for podcast
```
